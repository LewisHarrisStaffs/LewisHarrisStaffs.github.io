About Me

Level 4 Student at Staffordshire University studying for BSc (Hons) Computer Games Programming. I enjoy coding and have made several games for Game Jams, including the Lost cartridge Game Jam (Which focuses on NES-style Pixel art games) and the Staffordshire University Game Jam.

Portfolio

Below are some examples of my proudest peices of work. These include game jams and personal pet-projects.

S.P.A.C.E

This game was created for the Staffordshire University welcome week game jam (5 days).
It is a turn-based strategy game where the objective is to kill as many enemies as you can within 5 minutes.

Robo Rythm

Robo Rythm was a pet-project of mine. It is a rythm game, but the users get to create their own levels.
In this game, I coded a level editor, song selection, external file storage, particle system and more from scratch (using GameMaker Studio 2)
